## ğŸš€ Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/portfolio.git
cd portfolio
```

2. Open in browser:
```bash
# Simply open index.html in your browser
# Or use a local server
python -m http.server 8000
# Then navigate to http://localhost:8000
```

### Customization

1. **Update Personal Information:**
   - Replace email, phone, and social media links
   - Add your profile image
   - Update all placeholder text

2. **Add Your Content:**
   - Update job descriptions in experience.html
   - Update project details in projects.html
   - Fill in education information
   - Add extracurricular activities

3. **Modify Colors:**
   - Edit CSS variables in `src/style/style.css`
   - Update `:root` properties for brand colors

4. **Add Project Pages:**
   - Create new HTML files in `src/html/projects/`
   - Use pixelpaws.html as a template
   - Update links in projects.html
