PowerShell - Production Floor Monitor Script

Automated monitoring script that switches between web interface and PDF documents based on system status:

# Production Floor Monitoring System
$logPath = "C:\Logs\production.log"
$statusCheckInterval = 300  # 5 minutes

while($true) {
    try {
        $systemStatus = Get-SystemStatus
        $timestamp = Get-Date -Format "yyyy-MM-dd HH:mm:ss"
        
        if ($systemStatus.IsHealthy) {
            # Display main web interface
            Open-BrowserWindow -URL "http://intranet.lakeside.com/dashboard"
            Write-Log "System healthy - displaying dashboard"
        } else {
            # Display alert PDF with error details
            Open-AlertPDF -Status $systemStatus
            Write-Log "System alert - displaying error report: $($systemStatus.Error)"
        }
        
        # Log status and wait for next check
        Add-Content -Path $logPath -Value "$timestamp | Status: $($systemStatus.Status)"
        Start-Sleep -Seconds $statusCheckInterval
    }
    catch {
        Write-Log "Error in monitoring loop: $_"
    }
}

VBA - Database Automation Function

VBA function for automating data import and validation in Access databases:

Public Function AutomateDataImport(sourceFile As String, destinationTable As String) As Boolean
    On Error GoTo ErrorHandler
    
    Dim db As Database
    Dim rst As Recordset
    Dim fso As Object
    
    Set db = CurrentDb()
    Set fso = CreateObject("Scripting.FileSystemObject")
    
    ' Validate source file exists
    If Not fso.FileExists(sourceFile) Then
        MsgBox "Source file not found: " & sourceFile
        GoTo ErrorHandler
    End If
    
    ' Clear existing data
    db.Execute "DELETE FROM [" & destinationTable & "]"
    
    ' Import data with validation
    Set rst = db.OpenRecordset(destinationTable)
    
    ' Process data rows
    With rst
        .AddNew
        !FieldName = ValidateData(importedValue)
        .Update
    End With
    
    rst.Close
    AutomateDataImport = True
    Exit Function
    
ErrorHandler:
    MsgBox "Error: " & Err.Description
    AutomateDataImport = False
End Function

SQL - Database View for Reporting

SQL view for consolidating production data for Power BI reporting:

CREATE VIEW vwProductionDashboard AS
SELECT 
    p.ProductionID,
    p.ProductionDate,
    p.ItemCount,
    p.Status,
    d.DepartmentName,
    e.EmployeeName,
    CASE 
        WHEN p.Status = 'Complete' THEN 'On Schedule'
        WHEN p.Status = 'In Progress' THEN 'Active'
        WHEN DATEDIFF(DAY, p.ProductionDate, GETDATE()) > 2 THEN 'Overdue'
        ELSE 'Pending'
    END AS Priority,
    (SELECT COUNT(*) FROM ProductionErrors pe 
     WHERE pe.ProductionID = p.ProductionID) AS ErrorCount
FROM Production p
INNER JOIN Department d ON p.DepartmentID = d.DepartmentID
LEFT JOIN Employee e ON p.EmployeeID = e.EmployeeID
WHERE p.ProductionDate >= DATEADD(MONTH, -3, GETDATE())
ORDER BY p.ProductionDate DESC;
